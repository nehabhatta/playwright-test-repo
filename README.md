# project title: restful-booker-platform demo (hotel booking) testing

## Description
this involve testing a demo website for basic functionality of hotel booking such as room reservation.

## How to use
1. Clone the repository
2. Open the project
3. From the terminal install all the dependencies using 'npm i'

## Required Software
1. Node.js (version 16 or above)
2. Playwright testing framework
3. VS Code (optional, for editing and running tests)
